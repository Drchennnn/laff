一个 Windows 后台小工具：终端命令执行成功或失败时，自动播放对应音效。

- 命令成功 → 播放成功音效
- 命令失败 → 播放失败音效（比如罐头笑声）
- 每种事件支持多个音效，每次随机播一个
- 系统托盘图标，右键可开关
- 支持自定义音效包，换目录就换一套声音
- 同类事件 2 秒冷却，不会连续触发
